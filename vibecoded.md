## AI-Assisted Development Tools Used

This project was built with the assistance of the following AI-based development tools:

### 1. OpenAI Codex

* Used for generating backend logic
* Helped implement API routes and serverless handler structure
* Assisted in structuring MongoDB integration

### 2. Cursor IDE

* Used for frontend and backend development
* Assisted in writing React components and improving UI flow
* Helped debug integration between frontend and backend

### 3. Claude AI

* Used during later stages of development
* Assisted in resolving deployment issues on Vercel
* Helped fix API routing problems and environment configuration
* Provided minor UI improvements and refinements

---

## How AI Tools Were Used

* AI tools were used as **assistants**, not replacements for understanding
* All generated code was reviewed, tested, and modified as needed
* Debugging steps and deployment fixes required manual reasoning and iteration

---

## Key Areas Where AI Helped

* Backend API creation (Groq integration)
* Debugging serverless deployment issues (Vercel)
* Frontend UI structuring (React + Tailwind)
* Error handling and response formatting

The final implementation reflects a combination of AI assistance and manual problem-solving, especially during deployment, API debugging, and system integration.
