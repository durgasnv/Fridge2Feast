# Fridge2Feast 🍳

## Overview
AI-powered recipe generator using 3 ingredients.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- AI: Groq (llama-3.1-8b-instant)
- Database: MongoDB Atlas

## Features
- Single query → single response
- AI-generated recipe
- MongoDB storage

## Setup
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Deployment
- Frontend → Vercel
- Backend → Vercel

## API
POST /generate-recipe
